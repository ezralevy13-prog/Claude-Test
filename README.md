# NWS Bay Area Forecast App

A beautiful weather forecast application that provides daily summaries and detailed scientific forecast discussions from the National Weather Service Bay Area office.

## About

This repository contains a Dark Sky-inspired web application built with React, TypeScript, and Tailwind CSS. It displays National Weather Service (NWS) Area Forecast Discussions (AFD) from the Bay Area office in a clean, modern interface.

## Application

The main application is located in the `nws-forecast-app/` directory. See the [app README](./nws-forecast-app/README.md) for detailed documentation.

## Quick Start

```bash
cd nws-forecast-app
npm install
npm run dev
```

Visit `http://localhost:5173` to view the app.
